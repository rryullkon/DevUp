Déploiement d'une application React

Apprenez à déployer votre application sur Netlify, Vercel ou d'autres plateformes.

Chapitre 14: Déploiement d'une application React

1. Préparer votre application

Avant de déployer, construisez votre application en mode production avec la commande : 

npm run build

Cela génère un dossier build contenant les fichiers optimisés à déployer.

2. Déploiement sur Netlify

Netlify est une plateforme simple et gratuite pour déployer des sites statiques et applications React.

  1. Créez un compte sur Netlify.
  2. Connectez votre dépôt GitHub, GitLab ou Bitbucket.
  3. Configurez la branche à déployer (ex: main) et la commande de build (npm run build).
  4. Définissez le dossier de publication sur build.
  5. Lancez le déploiement, votre site sera accessible en quelques secondes.

3. Déploiement sur Vercel

Vercel est une autre plateforme populaire pour déployer facilement vos applications React.

  1. Créez un compte sur Vercel.
  2. Importez votre projet depuis GitHub, GitLab ou Bitbucket.
  3. Vercel détecte automatiquement les projets React et configure la build (npm run build) et le dossier de sortie (build).
  4. Démarrez le déploiement et votre application sera disponible rapidement.

4. Autres plateformes

Vous pouvez aussi déployer votre application sur :

  • GitHub Pages (pour des sites statiques)
  • Firebase Hosting
  • Amazon S3 + CloudFront
  • Heroku (avec Node.js pour SSR ou API)

5. Conseils pratiques

  • Configurez un fichier _redirects (Netlify) ou vercel.json pour gérer le routage côté client (SPA).
  • Utilisez un domaine personnalisé pour votre site.
  • Surveillez les performances et erreurs après déploiement.
  • Automatisez le déploiement avec les pipelines CI/CD de votre plateforme.
