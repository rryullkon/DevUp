Configuration de React

Installez Node.js, npm et configurez Create React App pour démarrer vos projets.

Chapitre 2: Configuration de l'environnement

1. Installer Node.js et npm

React nécessite Node.js et npm pour fonctionner.
Rendez-vous sur le site officiel : nodejs.org
Téléchargez la version LTS (Long Term Support) recommandée.
Lancez l'installateur et suivez les étapes par défaut.

Une fois l'installation terminée, ouvrez votre terminal et vérifiez les versions : 

node -v
npm -v

2. Installer Create React App

Create React App est un outil qui permet de démarrer rapidement un projet React sans configuration complexe.

Dans le terminal, tapez :

npm install -g create-react-app

Cela installe l'outil en global, pour l'utiliser n'importe où.

3. Créer un projet React

Une fois Create React App installé, vous pouvez créer un nouveau projet avec la commande : 

npx create-react-app mon-projet

Remplacez mon-projet par le nom de votre application.
Patientez pendant que les dépendances s’installent.

4. Lancer le projet

Pour démarrer le serveur de développement, utilisez les commandes suivantes : 

cd mon-projet
npm start

Cela ouvrira automatiquement votre navigateur sur http://localhost:3000 avec votre application React en fonctionnement.

5. Structure du projet

Voici la structure de base générée :

mon-projet/
├── node_modules/
├── public/
├── src/
│   └── App.js
├── package.json
└── README.md

6. Modifier le projet

Vous pouvez maintenant éditer le fichier src/App.js pour commencer à développer votre propre interface React.

Chaque fois que vous enregistrez une modification, l’application se recharge automatiquement.
