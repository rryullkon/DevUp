Les Hooks

Découvrez les Hooks (useState, useEffect, etc.) pour une gestion moderne de l'état et des effets.

Chapitre 7: Les Hooks

1. Qu'est-ce qu'un Hook ?

Les Hooks sont des fonctions introduites dans React 16.8 qui permettent d’utiliser l’état et d’autres fonctionnalités React sans écrire de classes.

2. useState

Le Hook useState permet de gérer l’état local dans un composant fonctionnel. 

import { useState } from 'react';

function Compteur() {
  const [compte, setCompte] = useState(0);

  return (
    <div>
      <p>Vous avez cliqué {compte} fois</p>
      <button onClick={() => setCompte(compte + 1)}>Cliquez-moi</button>
    </div>
  );
}

3. useEffect

Le Hook useEffect permet de gérer les effets de bord comme les appels API, les timers, ou les abonnements. 

import { useEffect } from 'react';

function ExempleEffet() {
  useEffect(() => {
    console.log('Composant monté');

    return () => {
      console.log('Composant démonté');
    };
  }, []);

  return <p>Regardez la console !</p>;
}

4. Autres Hooks utiles

  • useContext – Accéder à un contexte React
  • useRef – Référencer un élément DOM ou une valeur persistante
  • useReducer – Alternative à useState pour des états complexes
  • useMemo et useCallback – Optimisation des performances

5. Règles des Hooks

  • Appelez les Hooks uniquement au niveau supérieur de vos fonctions (pas dans des boucles ou des conditions).
  • Appelez-les uniquement dans des fonctions de composants ou des Hooks personnalisés.
  • Respectez l’ordre d’appel des Hooks entre les rendus.
