Création d'une application de tâches

Construisez une application de gestion de tâches complète avec React.

Travaux pratique - Chapitre 15 : Création d'une application de tâches

Objectif

Créer une application React permettant d’ajouter, afficher, compléter et supprimer des tâches. Ce projet vous permettra de mettre en pratique la gestion d’état, les événements, les listes et le rendu conditionnel.

Étapes détaillées

1. Initialisation du projet

Si ce n’est pas déjà fait, créez un nouveau projet React avec Create React App : 

npx create-react-app gestion-taches
cd gestion-taches
npm start

2. Structure de base du composant principal

Créez un composant App.js qui contiendra la liste des tâches et le formulaire d’ajout. 

import React, { useState } from 'react';

function App() {
  const [taches, setTaches] = useState([]);
  const [nouvelleTache, setNouvelleTache] = useState('');

  // Fonctions à compléter...

  return (
    <div>
      <h1>Gestionnaire de tâches</h1>
      {/* Formulaire d'ajout */}
      {/* Liste des tâches */}
    </div>
  );
}

export default App;

3. Ajouter une tâche

Implémentez un formulaire simple avec un champ texte et un bouton pour ajouter une tâche. Pensez à gérer la mise à jour de l’état lors de la saisie et l’ajout à la liste. 

function ajouterTache(e) {
  e.preventDefault();
  if (nouvelleTache.trim() === '') return;

  setTaches([...taches, { id: Date.now(), texte: nouvelleTache, complete: false }]);
  setNouvelleTache('');
}
        
<form onSubmit={ajouterTache}>
  <input
    type="text"
    value={nouvelleTache}
    onChange={e => setNouvelleTache(e.target.value)}
    placeholder="Ajouter une nouvelle tâche"
  />
  <button type="submit">Ajouter</button>
</form>

4. Afficher la liste des tâches

Affichez les tâches dans une liste. Chaque tâche doit montrer son texte et un bouton pour la supprimer. 

<ul>
  {taches.map(tache => (
    <li key={tache.id}>
      {tache.texte}
      <button onClick={() => supprimerTache(tache.id)}>Supprimer</button>
    </li>
  ))}
</ul>
        
function supprimerTache(id) {
  setTaches(taches.filter(tache => tache.id !== id));
}

5. Marquer une tâche comme complète

Ajoutez une case à cocher permettant de marquer une tâche comme terminée, et appliquez un style différent pour les tâches complètes. 

<li key={tache.id}>
  <input
    type="checkbox"
    checked={tache.complete}
    onChange={() => toggleComplete(tache.id)}
  />
  <span style={{ textDecoration: tache.complete ? 'line-through' : 'none' }}>
    {tache.texte}
  </span>
  <button onClick={() => supprimerTache(tache.id)}>Supprimer</button>
</li>
        
function toggleComplete(id) {
  setTaches(
    taches.map(tache =>
      tache.id === id ? { ...tache, complete: !tache.complete } : tache
    )
  );
}

6. Bonus : sauvegarde locale

Pour aller plus loin, sauvegardez la liste des tâches dans le localStorage afin de la conserver après actualisation de la page. 

import React, { useState, useEffect } from 'react';

function App() {
  const [taches, setTaches] = useState(() => {
    const saved = localStorage.getItem('taches');
    return saved ? JSON.parse(saved) : [];
  });

  useEffect(() => {
    localStorage.setItem('taches', JSON.stringify(taches));
  }, [taches]);

  // reste du code...
}

Conclusion

Ce projet simple vous permet de manipuler les concepts clés de React : gestion d’état avec useState, événements, rendu conditionnel et interaction utilisateur. N’hésitez pas à améliorer l’application en ajoutant par exemple la modification des tâches, des filtres ou une interface plus avancée.
