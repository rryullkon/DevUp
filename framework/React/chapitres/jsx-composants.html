Guide de démarrage de React avec JSX

Apprenez à utiliser JSX et créez vos premiers composants React.

Chapitre 3: JSX et les composants

1. Qu'est-ce que JSX ?

JSX (JavaScript XML) est une extension de syntaxe pour JavaScript qui permet d’écrire du HTML dans du code JavaScript. C’est utilisé avec React pour décrire l’interface utilisateur de manière déclarative.

Exemple simple :

const element = <h1>Bonjour, monde !</h1>;

Cette syntaxe sera transformée en code JavaScript équivalent à : 

const element = React.createElement('h1', null, 'Bonjour, monde !');

2. Créer un composant React

Un composant React est une fonction JavaScript qui retourne du JSX. Voici un exemple de composant simple : 

function Bonjour() {
  return <h1>Salut à tous !</h1>;
}

3. Utiliser un composant

Pour afficher un composant, vous l’utilisez comme une balise HTML personnalisée : 

function App() {
  return (
    <div>
      <Bonjour />
    </div>
  );
}

4. Composants dans des fichiers séparés

Vous pouvez créer un composant dans un fichier à part, par exemple Bonjour.js : 

// Bonjour.js
function Bonjour() {
  return <h1>Bonjour depuis un fichier séparé !</h1>;
}

export default Bonjour;

Et l’importer dans votre composant principal : 

// App.js
import Bonjour from './Bonjour';

function App() {
  return (
    <div>
      <Bonjour />
    </div>
  );
}

5. Règles à retenir

  • Les composants doivent commencer par une majuscule.
  • Chaque composant doit retourner un seul élément parent.
  • Utilisez className au lieu de class pour définir des classes CSS.
