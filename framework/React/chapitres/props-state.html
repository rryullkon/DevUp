Guide de démarrage avec React

Chapitre 4 : Props et State

Maîtrisez les props et le state en React

1. Les props : passer des données

Les props (propriétés) permettent de passer des données d’un composant parent à un composant enfant.

Exemple :

// Composant enfant
function Bonjour(props) {
  return <h1>Bonjour, {props.nom} !</h1>;
}

// Composant parent
function App() {
  return <Bonjour nom="Marie" />;
}

Dans cet exemple, le composant App envoie une prop nom au composant Bonjour.

2. Le state : gérer l’état interne

Le state est utilisé pour stocker des données internes à un composant. Il peut changer au cours du temps, par exemple suite à une interaction utilisateur.

Pour utiliser le state, on utilise le hook useState :

import { useState } from 'react';

function Compteur() {
  const [compte, setCompte] = useState(0);

  return (
    <div>
      <p>Valeur actuelle : {compte}</p>
      <button onClick={() => setCompte(compte + 1)}>
        Incrémenter
      </button>
    </div>
  );
}

Ici, compte est une variable d’état, initialisée à 0. Chaque clic sur le bouton incrémente la valeur.

3. Props vs State

Props State
Passées par le composant parent Gérées à l’intérieur du composant
Immutables (ne changent pas dans le composant enfant) Peut changer avec useState
Utilisées pour la communication entre composants Utilisées pour gérer l’interactivité

4. Exemple combiné : props + state

function Salut(props) {
  const [clics, setClics] = useState(0);

  return (
    <div>
      <h2>Bonjour {props.nom} !</h2>
      <p>Tu as cliqué {clics} fois</p>
      <button onClick={() => setClics(clics + 1)}>
        Clique-moi
      </button>
    </div>
  );
}

function App() {
  return <Salut nom="Jean" />;
}
