Formulaire en React

Maîtrisez la création et la gestion des formulaires avec React.

Chapitre 8: Formulaire en React

1. Introduction aux formulaires React

En React, les formulaires sont gérés via des composants contrôlés où l’état des champs est lié au state du composant.

2. Exemple simple de formulaire contrôlé

import { useState } from 'react';

function Formulaire() {
  const [nom, setNom] = useState('');

  function handleChange(event) {
    setNom(event.target.value);
  }

  function handleSubmit(event) {
    event.preventDefault();
    alert('Nom soumis : ' + nom);
  }

  return (
    <form onSubmit={handleSubmit}>
      <label>
        Nom :
        <input type="text" value={nom} onChange={handleChange} />
      </label>
      <button type="submit">Envoyer</button>
    </form>
  );
}

3. Gestion de plusieurs champs

Pour plusieurs champs, vous pouvez utiliser un state objet et une fonction générique de mise à jour. 

function FormulaireMulti() {
  const [formData, setFormData] = useState({
    email: '',
    motDePasse: ''
  });

  function handleChange(event) {
    const { name, value } = event.target;
    setFormData(prevData => ({
      ...prevData,
      [name]: value
    }));
  }

  function handleSubmit(event) {
    event.preventDefault();
    alert('Données soumises : ' + JSON.stringify(formData));
  }

  return (
    <form onSubmit={handleSubmit}>
      <label>
        Email :
        <input
          type="email"
          name="email"
          value={formData.email}
          onChange={handleChange}
        />
      </label>
      <label>
        Mot de passe :
        <input
          type="password"
          name="motDePasse"
          value={formData.motDePasse}
          onChange={handleChange}
        />
      </label>
      <button type="submit">Connexion</button>
    </form>
  );
}

4. Bonnes pratiques

  • Utilisez des composants contrôlés pour garder le contrôle des données.
  • Validez les données avant la soumission.
  • Utilisez des noms d’attributs cohérents dans vos inputs.
  • Évitez la duplication de code avec des gestionnaires génériques.
