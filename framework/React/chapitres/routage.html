Routage avec React Router

Apprenez à implémenter la navigation entre pages avec React Router.

Chapitre 9: Routage avec React Router

1. Introduction

React Router est une bibliothèque qui permet de gérer la navigation dans une application React en créant des routes.

2. Installation

npm install react-router-dom

3. Configuration basique

import { BrowserRouter as Router, Routes, Route, Link } from 'react-router-dom';

function Accueil() {
  return <h2>Page d'accueil</h2>;
}

function APropos() {
  return <h2>À propos</h2>;
}

function App() {
  return (
    <Router>
      <nav>
        <Link to="/">Accueil</Link> | 
        <Link to="/apropos">À propos</Link>
      </nav>
      <Routes>
        <Route path="/" element={<Accueil />} />
        <Route path="/apropos" element={<APropos />} />
      </Routes>
    </Router>
  );
}

4. Concepts clés

  • Router : conteneur principal qui active la navigation.
  • Routes : conteneur pour définir les différentes routes.
  • Route : définit un chemin et le composant à afficher.
  • Link : remplace la balise <a> pour une navigation sans rechargement.

5. Navigation programmée

import { useNavigate } from 'react-router-dom';

function MonComposant() {
  const navigate = useNavigate();

  function allerAccueil() {
    navigate('/');
  }

  return <button onClick={allerAccueil}>Retour à l'accueil</button>;
}

6. Bonnes pratiques

  • Organisez vos routes dans des fichiers séparés si elles sont nombreuses.
  • Utilisez des routes imbriquées pour une meilleure hiérarchie.
  • Testez la navigation pour s’assurer que les liens fonctionnent correctement.
