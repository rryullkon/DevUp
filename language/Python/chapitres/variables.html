Variables et types de données en Python

Comprendre les variables, les types de données de base et leur manipulation

Chapitre 3: Variables et types de données

1. Qu'est-ce qu'une variable ?

Une variable est un conteneur pour stocker des données en mémoire.
En Python, on crée une variable simplement en lui attribuant une valeur : 

age = 25
nom = "Alice"

Python est dynamiquement typé : le type est déterminé automatiquement.

2. Types de données de base

Type Description Exemple
int Nombre entier 42
float Nombre à virgule flottante 3.14
str Chaîne de caractères "Bonjour"
bool Booléen (Vrai/Faux) True ou False
list Liste (collection ordonnée) [1, 2, 3]
dict Dictionnaire (paires clé-valeur) {"nom": "Alice", "age": 25}

3. Manipulation des variables

Affectation multiple

a, b, c = 5, 3.2, "Hello"

Conversion de type (casting)

nombre = "123"
nombre_entier = int(nombre)  # Convertit en entier

pi = 3.14
pi_texte = str(pi)  # Convertit en chaîne

Vérification du type

type(variable)  # Retourne le type de la variable

4. Bonnes pratiques

  • Noms de variables descriptifs (age_utilisateur plutôt que a)
  • Utilisez des underscores pour les noms composés (nom_complet)
  • Évitez les mots-clés Python (print, if, etc.)
  • Python est sensible à la casse (Age et age sont différents)
