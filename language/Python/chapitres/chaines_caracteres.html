Les chaînes de caractères en Python

Manipuler et utiliser les chaînes de caractères efficacement

Chapitre 4: Les chaînes de caractères

1. Définition et création

En Python, les chaînes (type str) sont des séquences de caractères entourées de guillemets simples ou doubles : 

chaine1 = 'Bonjour'
chaine2 = "Python"
chaine3 = '''Ceci est une
chaîne sur plusieurs
lignes'''

2. Opérations de base

Concaténation

prenom = "Alice"
nom = "Dupont"
nom_complet = prenom + " " + nom  # "Alice Dupont"

Répétition

separateur = "-" * 20  # "--------------------"

Accès aux caractères

langage = "Python"
premiere_lettre = langage[0]  # "P" (indexation à partir de 0)
derniere_lettre = langage[-1] # "n"

3. Méthodes utiles

Méthode Description Exemple
.upper() Convertit en majuscules "hello".upper() → "HELLO"
.lower() Convertit en minuscules "HELLO".lower() → "hello"
.strip() Supprime les espaces en début/fin " text ".strip() → "text"
.split() Découpe la chaîne en liste "a,b,c".split(",") → ["a","b","c"]
.replace() Remplace une sous-chaîne "hello".replace("e","a") → "hallo"
.format() Formatage de chaîne "Bonjour {}".format("Alice") → "Bonjour Alice"

4. Les f-strings (Python 3.6+)

Méthode moderne de formatage des chaînes : 

nom = "Alice"
age = 25
message = f"Je m'appelle {nom} et j'ai {age} ans"
# Résultat: "Je m'appelle Alice et j'ai 25 ans"
