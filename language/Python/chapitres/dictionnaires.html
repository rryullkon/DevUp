Les dictionnaires

Utiliser les dictionnaires pour stocker des paires clé/valeur.

Chapitre 12 : Les dictionnaires

1. Introduction

Un dictionnaire est une collection non ordonnée de paires clé/valeur. Chaque clé unique permet d’accéder à sa valeur associée, ce qui est très utile pour organiser et retrouver rapidement des données.

2. Créer un dictionnaire

On crée un dictionnaire en plaçant les paires clé/valeur entre accolades, séparées par des virgules. Les clés et valeurs sont séparées par deux-points. 

personne = {
    "nom": "Dupont",
    "age": 30,
    "ville": "Paris"
}

3. Accéder aux valeurs

On accède à la valeur associée à une clé en utilisant la clé entre crochets. 

print(personne["nom"])  # Affiche : Dupont
print(personne["age"])  # Affiche : 30

4. Modifier et ajouter des éléments

On peut modifier la valeur d’une clé existante ou en ajouter de nouvelles très simplement. 

personne["age"] = 31          # Modification
personne["profession"] = "Ingénieur"  # Ajout d'une nouvelle paire clé/valeur

5. Supprimer une paire clé/valeur

Pour supprimer une entrée, on utilise del avec la clé. 

del personne["ville"]

6. Parcourir un dictionnaire

On peut parcourir les clés, les valeurs ou les deux en même temps avec des boucles. 

# Parcourir les clés
for cle in personne:
    print(cle)

# Parcourir les valeurs
for valeur in personne.values():
    print(valeur)

# Parcourir clés et valeurs
for cle, valeur in personne.items():
    print(cle, ":", valeur)

7. Méthodes utiles

  • keys() : retourne une vue des clés
  • values() : retourne une vue des valeurs
  • items() : retourne une vue des paires clé/valeur
  • get(clé, valeur_par_defaut) : retourne la valeur associée à la clé ou la valeur par défaut si la clé n’existe pas
  • clear() : vide le dictionnaire

8. Exemple complet

contacts = {
    "Alice": "alice@example.com",
    "Bob": "bob@example.com"
}

# Ajouter un contact
contacts["Charlie"] = "charlie@example.com"

# Afficher tous les contacts
for nom, email in contacts.items():
    print(nom, "->", email)
