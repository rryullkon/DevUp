Installation de Python sur macOS

Guide complet pour installer Python correctement sur votre système Mac

Étape 1 : Vérification initiale

macOS inclut parfois une ancienne version de Python (2.x). Vérifions d'abord :

python3 --version
which python3

Si vous obtenez une version 3.x, Python est déjà installé. Sinon, passez à l'étape suivante.

Étape 2 : Installation de Python

Deux options s'offrent à vous :

Option 1 : Utiliser Homebrew (recommandé)

# Installer Homebrew si ce n'est pas déjà fait
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Installer Python
brew install python

Option 2 : Télécharger depuis python.org

Étape 1 : Téléchargement

Pour installer Python sur macOS, suivez ces étapes :

Note importante : Téléchargez toujours Python depuis le site officiel pour éviter les versions modifiées ou malveillantes.
1

Rendez-vous sur python.org/downloads/mac-osx

2

Téléchargez le fichier .pkg correspondant à la dernière version de Python

3

Ouvrez le fichier téléchargé et suivez les instructions d'installation

Attention : Si une version de Python est déjà installée sur votre Mac, l’installeur pourra vous proposer une mise à jour ou une installation parallèle.

Étape 3 : Vérification

Vérifiez que Python et pip sont bien installés :

Terminal
$ python3 --version
Python 3.12.1
$ pip3 --version
pip 23.x from /usr/local/lib/python3.12/site-packages (python 3.12)

Dépannage

Installez Python avec Homebrew :

brew install python

Assurez-vous qu'il a été installé avec Python. Sinon, utilisez :

sudo easy_install pip
