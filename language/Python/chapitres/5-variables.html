1

2

3

Étape 2 : Affichage d'une variable

1

Utilisez la fonction print()

2

Par exemple : print(nom) affichera Alice

Étape 3 : Types de variables

Voici les types de variables les plus courants en Python :

Type Exemple Description
str nom = "Alice" Chaîne de caractères (texte).
int age = 30 Nombre entier sans virgule.
float taille = 1.75 Nombre à virgule flottante (décimal).
bool est_majeur = True Valeur booléenne : True ou False.

Dépannage

Assurez-vous que la variable est bien déclarée avant son utilisation.

# Mauvais
print(prenom)

# Correct
prenom = "Emma"
print(prenom)

Ne mélangez pas types incompatibles sans conversion :

# Mauvais :
age = 25
print("Âge : " + age)  # Erreur

# Bon :
print("Âge : " + str(age))
