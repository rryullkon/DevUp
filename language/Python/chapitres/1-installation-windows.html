Étape 1 : Téléchargement

Pour installer Python sur Windows, suivez ces étapes :

Note importante : Téléchargez toujours Python depuis le site officiel pour éviter les versions modifiées ou malveillantes.
1

Rendez-vous sur python.org/downloads

2

Cliquez sur le bouton "Download Python" (version la plus récente)

3

Enregistrez le fichier .exe sur votre ordinateur

Attention : Selon votre système, choisissez la version 32-bit ou 64-bit. Pour vérifier :
  • Faites un clic droit sur "Ce PC" > Propriétés
  • Regardez la section "Type de système"

Étape 2 : Lancement de l'installateur

Après le téléchargement :

1

Double-cliquez sur le fichier téléchargé (ex: python-3.9.7.exe)

2

Cochez la case "Add Python to PATH" (très important !)

3

Cliquez sur "Install Now"

# Ceci est un exemple de code qu'on pourrait voir plus tard
python --version  # Vérifie la version installée
pip list          # Liste les packages installés

Étape 3 : Configuration

Options recommandées :

Option Recommandation Description
Install launcher for all users ✓ Permet à tous les utilisateurs d'accéder à Python
Associate files with Python ✓ Ouvre les fichiers .py avec Python par défaut
Create shortcuts Optionnel Crée des raccourcis dans le menu Démarrer
Add Python to PATH Obligatoire Permet d'utiliser Python depuis n'importe quel dossier

Étape 4 : Vérification

Après installation, vérifiez que tout fonctionne :

Command Prompt
C:\> python --version
Python 3.9.7
C:\> pip --version
pip 21.2.4 from C:\Python39\lib\site-packages\pip (python 3.9)

Si vous voyez les versions s'afficher comme ci-dessus, Python est correctement installé !

Dépannage

Cela signifie que Python n'a pas été ajouté au PATH. Solutions :

  1. Réinstaller Python en cochant "Add Python to PATH"
  2. Ou ajouter manuellement le chemin Python au PATH système
setx PATH "%PATH%;C:\Python39"

Essayez ces solutions :

  1. Téléchargez à nouveau le fichier d'installation
  2. Vérifiez que votre antivirus ne bloque pas l'installation
  3. Essayez avec une version légèrement plus ancienne de Python
