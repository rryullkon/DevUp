Les fonctions

Définir et utiliser des fonctions pour structurer votre code.

Chapitre 9 : Les fonctions

1. Introduction

Une fonction est un bloc de code réutilisable qui permet d’exécuter une tâche spécifique. Les fonctions facilitent la lecture, la maintenance et l’organisation du code.

2. Définir une fonction

En Python, on crée une fonction avec le mot-clé def, suivi du nom de la fonction et des parenthèses. Le corps de la fonction est indenté. 

def saluer():
    print("Bonjour !")

3. Appeler une fonction

Pour exécuter le code d’une fonction, il suffit de l’appeler par son nom suivi de parenthèses. 

saluer()  # Affiche : Bonjour !

4. Les paramètres

Les fonctions peuvent recevoir des informations via des paramètres pour rendre leur comportement flexible. 

def saluer_personne(nom):
    print("Bonjour", nom)

saluer_personne("Alice")  # Affiche : Bonjour Alice

5. La valeur de retour

Une fonction peut renvoyer un résultat avec l’instruction return. Cela permet de récupérer une valeur à la fin de son exécution. 

def carre(x):
    return x * x

resultat = carre(5)
print(resultat)  # Affiche : 25

6. Paramètres optionnels

On peut définir des paramètres avec des valeurs par défaut, qui seront utilisées si aucun argument n’est passé lors de l’appel. 

def saluer_personne(nom="inconnu"):
    print("Bonjour", nom)

saluer_personne()          # Affiche : Bonjour inconnu
saluer_personne("Bob")     # Affiche : Bonjour Bob

7. Portée des variables

Les variables définies à l’intérieur d’une fonction sont locales à cette fonction. Elles ne sont pas accessibles à l’extérieur. 

def ma_fonction():
    message = "Salut"
    print(message)

ma_fonction()  # Affiche : Salut
# print(message)  # Provoquerait une erreur : variable non définie

8. Exemple complet

def calculer_somme(a, b):
    return a + b

resultat = calculer_somme(10, 20)
print("Somme =", resultat)  # Somme = 30
