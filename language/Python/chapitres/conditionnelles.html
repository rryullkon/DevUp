Les structures conditionnelles

Contrôler le flux du programme avec if, elif et else.

Chapitre 7 : Les structures conditionnelles

1. Introduction

Les structures conditionnelles permettent à un programme de **prendre des décisions** en fonction de certaines conditions. Elles sont essentielles pour contrôler le flux d'exécution du code.

2. La structure if

La condition if exécute un bloc de code seulement si une expression est évaluée à True. 

age = 18
if age >= 18:
    print("Vous êtes majeur")  # Ce message s'affiche

3. Le mot-clé else

Le bloc else est exécuté uniquement si la condition du if est False. 

age = 16
if age >= 18:
    print("Vous êtes majeur")
else:
    print("Vous êtes mineur")  # Ce message s'affiche

4. La condition elif

elif (abréviation de "else if") permet de tester plusieurs conditions de manière ordonnée. Une seule branche sera exécutée : la première dont la condition est vraie. 

note = 15

if note >= 16:
    print("Très bien")
elif note >= 14:
    print("Bien")  # Ce message s'affiche
elif note >= 12:
    print("Assez bien")
else:
    print("Peut mieux faire")

5. Indentation en Python

En Python, l’indentation (les espaces ou tabulations au début d’une ligne) est **obligatoire** pour délimiter les blocs de code. Une erreur d’indentation provoquera une erreur d’exécution. 

# Correct :
if True:
    print("Ceci est indenté correctement")

# Incorrect :
if True:
print("Erreur : mauvaise indentation")  # provoque une erreur

6. Conditions combinées

On peut combiner plusieurs conditions grâce aux opérateurs logiques and, or et not. 

age = 20
nationalite = "française"

if age >= 18 and nationalite == "française":
    print("Vous pouvez voter en France")

7. Expressions booléennes

Toute expression qui renvoie une valeur de type bool peut être utilisée dans une condition. Par exemple : 

nom = ""

if not nom:
    print("Le nom est vide")

8. Exemples pratiques

Voici un exemple plus complet : 

temperature = 30

if temperature > 35:
    print("Il fait très chaud !")
elif temperature > 25:
    print("Il fait chaud.")  # Ce message s'affiche
elif temperature > 15:
    print("Il fait bon.")
else:
    print("Il fait frais ou froid.")

Ces structures rendent votre programme plus **intelligent** et **réactif** selon différentes situations.
