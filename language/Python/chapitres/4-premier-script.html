Votre premier script Python

Guide pour écrire, sauvegarder et exécuter votre premier script Python

Sommaire du chapitre

  • Étape 1 : Créer un fichier Python
  • Étape 2 : Écrire le script
  • Étape 3 : Exécuter le script
  • Dépannage

Étape 1 : Créer un fichier Python

1

Ouvrez le Bloc-notes ou un éditeur de code comme VS Code ou Notepad++

2

Sauvegardez le fichier sous le nom mon_script.py

Conseil : Assurez-vous que l'extension soit bien .py et non .txt.

Étape 2 : Écrire le script

Dans le fichier mon_script.py, écrivez ce code :

# Ceci est votre premier script Python
print("Bonjour, Python !")

Étape 3 : Exécuter le script

1

Ouvrez l’invite de commande (cmd)

2

Utilisez la commande cd pour naviguer jusqu’au dossier contenant mon_script.py

3

Lancez la commande suivante :

python mon_script.py
Command Prompt
C:\Utilisateurs\Nom\Documents> python mon_script.py
Bonjour, Python !

Dépannage

Vérifiez que Python est bien ajouté au PATH :

  • Réinstallez Python en cochant "Add Python to PATH"
  • Ou ajoutez manuellement le dossier Python au PATH système

Assurez-vous que :

1

Le fichier est bien enregistré avec l’extension .py

2

Vous êtes dans le bon répertoire dans le terminal
