Chapitre 1: Introduction à Python

1. Un langage polyvalent

Python est un langage de programmation interprété, haut niveau et multi-paradigme.
Créé en 1991 par Guido van Rossum, il est aujourd'hui l'un des langages les plus populaires au monde.

2. À quoi sert Python ?

Python est utilisé dans de nombreux domaines :

  • Développement web (Django, Flask)
  • Analyse de données (Pandas, NumPy)
  • Intelligence artificielle (TensorFlow, PyTorch)
  • Automatisation de tâches
  • Scripting système
  • Jeux vidéo (Pygame)

3. Pourquoi choisir Python ?

Les avantages de Python :

  1. Syntaxe claire et lisible (moins de code qu'en Java ou C++)
  2. Grande communauté et nombreuses bibliothèques
  3. Multiplateforme (Windows, macOS, Linux)
  4. Idéal pour les débutants tout en étant puissant

4. Exemple de code Python

Voici un simple programme qui affiche "Bonjour le monde" et calcule la somme de deux nombres : 

# Ceci est un commentaire
print("Bonjour le monde !")

a = 5
b = 3
somme = a + b
print(f"La somme de {a} et {b} vaut {somme}")

5. Premiers pas

Pour commencer avec Python :

  1. Installez Python sur votre ordinateur
  2. Utilisez un éditeur de code (VS Code, PyCharm, etc.)
  3. Consultez les tutoriels pour apprendre les bases
