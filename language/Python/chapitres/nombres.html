Les nombres : entiers et flottants

Découvrir les types numériques et leurs opérations.

Chapitre 5 : Les nombres – entiers et flottants

1. Types numériques en Python

Python propose différents types pour représenter les nombres :

  • int : pour les entiers (positifs ou négatifs).
  • float : pour les nombres à virgule flottante (décimaux).
entier = 10           # int
flottant = 3.14       # float

2. Opérations de base

On peut effectuer des opérations arithmétiques de base sur les nombres : 

a = 10
b = 3

addition = a + b        # 13
soustraction = a - b    # 7
multiplication = a * b  # 30
division = a / b        # 3.333...
division_entiere = a // b  # 3
modulo = a % b          # 1 (reste de la division)
puissance = a ** b      # 1000 (10 puissance 3)

3. Conversion de types

Il est possible de convertir un entier en flottant et vice-versa : 

int(3.9)     # 3
float(7)     # 7.0

4. Fonctions utiles

Python propose plusieurs fonctions pour travailler avec les nombres :

  • abs(x) : valeur absolue
  • round(x) : arrondi au plus proche
  • pow(x, y) : puissance (équivalent à x ** y)
  • min(x, y, ...) et max(x, y, ...) : minimum et maximum
abs(-5)        # 5
round(3.14159, 2)  # 3.14
pow(2, 3)      # 8
min(3, 7, -2)  # -2
max(3, 7, -2)  # 7

5. Particularités des flottants

Les nombres flottants peuvent avoir des imprécisions dues à leur représentation en mémoire : 

0.1 + 0.2  # 0.30000000000000004

Pour des calculs financiers ou plus précis, on peut utiliser le module decimal : 

from decimal import Decimal
resultat = Decimal("0.1") + Decimal("0.2")  # 0.3
