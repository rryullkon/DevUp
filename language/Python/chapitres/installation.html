Guide d'installation de Python

Un tutoriel complet pour configurer Python sur votre système Windows

Installation de Python sur Windows

1. Télécharger Python

Rendez-vous sur le site officiel de Python : python.org/downloads.
Cliquez sur le bouton "Download Python" (la version la plus récente est recommandée).
Téléchargez l'installeur adapté à votre système (Windows).

2. Lancer l'installation

  1. Ouvrez le fichier téléchargé (ex : python-3.x.x.exe).
  2. Important : Cochez la case "Add Python to PATH" avant de cliquer sur "Install Now".
  3. Cliquez sur "Install Now" et laissez l'installation se terminer.

3. Vérifier l'installation

Ouvrez l'invite de commande (CMD) et tapez la commande suivante : 

python --version

Si Python est installé, la version s'affichera (ex : Python 3.12.0).
Si la commande n'est pas reconnue, redémarrez votre ordinateur ou vérifiez que Python est bien ajouté au PATH.

4. Installer pip (gestionnaire de paquets)

Pip est normalement installé automatiquement avec Python. Pour vérifier : 

pip --version

Si la version de pip s'affiche, tout est prêt.
Sinon, consultez la documentation officielle : Installer pip

5. Tester Python

Créez un fichier test.py avec le code suivant : 

print("Python fonctionne !")

Exécutez-le avec la commande python test.py dans le terminal.
Si le message s'affiche, Python est bien installé.
