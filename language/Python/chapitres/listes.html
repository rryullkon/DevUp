Les listes

Créer, modifier et parcourir des listes en Python.

Chapitre 10 : Les listes

1. Introduction

Une liste est une collection ordonnée et modifiable d’éléments. Elle peut contenir des éléments de différents types (nombres, chaînes, objets, etc.).

2. Créer une liste

On crée une liste en plaçant les éléments entre crochets, séparés par des virgules. 

fruits = ["pomme", "banane", "cerise"]
nombres = [1, 2, 3, 4, 5]
liste_vide = []

3. Accéder aux éléments

Les éléments d’une liste sont accessibles par leur indice, en commençant à 0. 

fruits = ["pomme", "banane", "cerise"]
print(fruits[0])  # Affiche : pomme
print(fruits[-1]) # Affiche : cerise (dernier élément)

4. Modifier une liste

Les listes sont modifiables, on peut changer, ajouter ou supprimer des éléments. 

fruits = ["pomme", "banane", "cerise"]
fruits[1] = "orange"         # Remplace "banane" par "orange"
fruits.append("kiwi")        # Ajoute "kiwi" à la fin
fruits.remove("pomme")       # Supprime "pomme"
print(fruits)                # Affiche : ['orange', 'cerise', 'kiwi']

5. Parcourir une liste

La boucle for est idéale pour parcourir les éléments d’une liste. 

fruits = ["orange", "cerise", "kiwi"]
for fruit in fruits:
    print(fruit)

6. Quelques méthodes utiles

Voici quelques méthodes courantes pour manipuler les listes :

  • append(x) : ajoute l’élément x à la fin
  • insert(i, x) : insère x à la position i
  • pop(i) : enlève et renvoie l’élément à l’indice i (par défaut le dernier)
  • sort() : trie la liste
  • reverse() : inverse l’ordre des éléments
  • len() : retourne la taille de la liste

7. Exemples pratiques

Ajouter des nombres et afficher leur somme :

nombres = [1, 2, 3, 4, 5]
total = 0
for n in nombres:
    total += n
print("Total =", total)  # Total = 15

Créer une liste vide et la remplir avec une boucle :

carres = []
for i in range(1, 6):
    carres.append(i * i)
print(carres)  # Affiche : [1, 4, 9, 16, 25]
