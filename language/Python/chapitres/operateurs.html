Les opérateurs en Python

Utiliser les opérateurs arithmétiques, logiques et de comparaison.

Chapitre 6 : Les opérateurs en Python

1. Les opérateurs arithmétiques

Ces opérateurs permettent d’effectuer des calculs mathématiques :

Opérateur Nom Exemple Résultat
+ Addition 5 + 3 8
- Soustraction 5 - 2 3
* Multiplication 4 * 3 12
/ Division 10 / 4 2.5
// Division entière 10 // 3 3
% Modulo (reste) 10 % 3 1
** Puissance 2 ** 3 8

2. Les opérateurs de comparaison

Ils sont utilisés pour comparer deux valeurs. Le résultat est un booléen (True ou False) : 

x = 5
y = 3

x == y   # False (égal à)
x != y   # True (différent de)
x > y    # True (supérieur à)
x < y    # False (inférieur à)
x >= y   # True (supérieur ou égal à)
x <= y   # False (inférieur ou égal à)

3. Les opérateurs logiques

Ils permettent de combiner plusieurs conditions : 

a = True
b = False

a and b   # False (ET logique)
a or b    # True (OU logique)
not a     # False (NON logique)

4. Combinaison d'opérateurs

On peut combiner les opérateurs pour construire des expressions plus complexes : 

age = 20
citoyen = True

if age >= 18 and citoyen:
    print("Peut voter")  # Cette condition est vraie

5. Priorité des opérateurs

Comme en mathématiques, les expressions respectent un ordre de priorité. L’usage des parenthèses permet de clarifier : 

resultat = 3 + 4 * 2    # 11 (multiplication d'abord)
resultat = (3 + 4) * 2  # 14 (addition d'abord grâce aux parenthèses)
