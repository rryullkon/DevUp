Installation de Python sur Linux

Guide complet pour installer Python correctement sur votre système Linux

Sommaire du chapitre

  • Étape 1 : Mise à jour du système
  • Étape 2 : Installation de Python
  • Étape 3 : Vérification
  • Dépannage

Étape 1 : Mise à jour du système

Avant toute chose, assurez-vous que votre système est à jour :

sudo apt update && sudo apt upgrade -y   # Pour les distributions basées sur Debian/Ubuntu
sudo dnf update -y                        # Pour Fedora
sudo pacman -Syu                          # Pour Arch Linux

Étape 2 : Installation de Python

La commande varie selon votre distribution :

1

Ubuntu/Debian : sudo apt install python3 python3-pip -y

2

Fedora : sudo dnf install python3 -y

3

Arch Linux : sudo pacman -S python

Étape 3 : Vérification

Vérifiez que Python et pip sont bien installés :

Terminal
$ python3 --version
Python 3.11.x
$ pip3 --version
pip 23.x.x from /usr/lib/python3/dist-packages (python 3.11)

Dépannage

Python n'est peut-être pas installé ou votre PATH est mal configuré.

  1. Réinstallez Python avec la commande adaptée à votre distribution
  2. Vérifiez avec which python3 ou echo $PATH

Essayez :

  1. sudo apt install python3-pip (Ubuntu/Debian)
  2. Ou installez manuellement via curl https://bootstrap.pypa.io/get-pip.py | sudo python3
