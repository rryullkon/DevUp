Les boucles for et while

Répéter des instructions avec les boucles for et while.

Chapitre 8 : Les boucles for et while

1. Introduction

Les boucles sont des structures qui permettent de répéter un bloc de code plusieurs fois, ce qui est essentiel pour automatiser des tâches répétitives.

2. La boucle for

La boucle for est utilisée pour itérer sur une séquence (liste, chaîne de caractères, plage de nombres, etc.). 

for i in range(5):
    print(i)
# Affiche : 0, 1, 2, 3, 4

Ici, range(5) crée une séquence de nombres de 0 à 4, et la boucle exécute le bloc print(i) pour chaque valeur.

3. Parcourir une liste avec for

fruits = ["pomme", "banane", "cerise"]
for fruit in fruits:
    print(fruit)
# Affiche chaque fruit dans la liste

4. La boucle while

La boucle while répète un bloc tant qu'une condition est vraie. 

compteur = 0
while compteur < 5:
    print(compteur)
    compteur += 1
# Affiche : 0, 1, 2, 3, 4

5. Différences entre for et while

  • for est idéal quand on connaît à l’avance le nombre d’itérations (ex: parcourir une liste).
  • while est utile quand la répétition dépend d’une condition qui peut changer durant l’exécution.

6. Les instructions break et continue

- break permet d’arrêter la boucle immédiatement.
- continue saute le reste du bloc et passe à l’itération suivante. 

for i in range(10):
    if i == 5:
        break  # Arrête la boucle quand i vaut 5
    print(i)

for i in range(5):
    if i == 2:
        continue  # Ignore l’itération quand i vaut 2
    print(i)

7. Exemples pratiques

Calculer la somme des nombres de 1 à 10 avec une boucle for : 

somme = 0
for nombre in range(1, 11):
    somme += nombre
print("Somme =", somme)  # Somme = 55

Compter à rebours avec une boucle while : 

compteur = 5
while compteur > 0:
    print(compteur)
    compteur -= 1
print("Décollage !")
