Les tuples

Découvrir les tuples, structures de données immuables.

Chapitre 11 : Les tuples

1. Introduction

Un tuple est une collection ordonnée d’éléments, similaire à une liste, mais **immuable**, ce qui signifie qu’on ne peut pas modifier ses éléments après sa création.

2. Créer un tuple

On crée un tuple en plaçant les éléments entre parenthèses, séparés par des virgules. 

coordonnees = (10, 20)
couleurs = ("rouge", "vert", "bleu")
tuple_vide = ()

3. Accéder aux éléments

Comme les listes, les éléments d’un tuple sont accessibles par leur indice, en commençant à 0. 

couleurs = ("rouge", "vert", "bleu")
print(couleurs[0])  # Affiche : rouge
print(couleurs[-1]) # Affiche : bleu

4. Immuabilité

Contrairement aux listes, on ne peut pas modifier, ajouter ou supprimer un élément dans un tuple après sa création. Toute tentative de modification provoquera une erreur. 

coordonnees = (10, 20)
# coordonnees[0] = 15  # Erreur : les tuples sont immuables

5. Utilité des tuples

Les tuples sont utiles lorsqu’on veut s’assurer que des données ne seront pas modifiées par erreur, ou pour représenter des données fixes comme des coordonnées géographiques.

6. Tuple avec un seul élément

Pour créer un tuple contenant un seul élément, il faut mettre une virgule après l’élément. 

mon_tuple = (5,)
print(type(mon_tuple))  # Affiche : <class 'tuple'>

7. Déballage de tuple (unpacking)

On peut assigner chaque élément d’un tuple à une variable distincte en une seule opération. 

point = (3, 7)
x, y = point
print(x)  # 3
print(y)  # 7

8. Conversion entre listes et tuples

On peut convertir une liste en tuple et inversement avec les fonctions tuple() et list(). 

ma_liste = [1, 2, 3]
mon_tuple = tuple(ma_liste)
print(mon_tuple)  # (1, 2, 3)

nouvelle_liste = list(mon_tuple)
print(nouvelle_liste)  # [1, 2, 3]
