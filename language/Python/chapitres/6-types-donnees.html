Étape 1 : Que sont les types de données ?

1

Un type de donnée indique à Python le type de valeur utilisée (texte, nombre, liste, etc.).

2

Chaque variable en Python a un type associé automatiquement dès sa création.

Étape 2 : Tableau des types courants

Type Exemple Description
str "Bonjour" Chaîne de caractères (texte)
int 42 Nombre entier
float 3.14 Nombre décimal (flottant)
bool True / False Booléen (vrai ou faux)
list [1, 2, 3] Liste ordonnée et modifiable
tuple (1, 2, 3) Liste ordonnée et non modifiable
dict {"nom": "Alice"} Dictionnaire (paires clé-valeur)
