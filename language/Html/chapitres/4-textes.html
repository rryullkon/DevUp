Étape 1 : Les en-têtes HTML

1

Les en-têtes HTML vont de <h1> à <h6>.

2

<h1> est le plus important, <h6> le moins important.

Étape 2 : Le paragraphe en HTML

Le texte courant est généralement placé dans des balises <p>.

<p>Ceci est un paragraphe de texte.</p>

Chaque paragraphe est affiché sur une nouvelle ligne avec une marge verticale par défaut.

Étape 3 : Hiérarchie et sémantique

Il est important de respecter la hiérarchie des en-têtes pour la structure logique de la page.


<h1>Titre principal</h1>
<h2>Sous-titre</h2>
<h3>Sous-section</h3>

Cette hiérarchie aide les moteurs de recherche et les technologies d’assistance à comprendre votre contenu.

Dépannage

Assurez-vous que votre paragraphe ou en-tête est bien encadré par les bonnes balises :

<p>Texte</p>
<h2>Titre</h2>

Les styles par défaut peuvent varier selon le navigateur. Utilisez du CSS pour personnaliser l’apparence des titres si nécessaire.
