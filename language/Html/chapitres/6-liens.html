Liens hypertextes

Apprenez à créer des liens vers d'autres pages ou sites web avec HTML

Sommaire du chapitre

  • Étape 1 : Lien vers un site externe
  • Étape 2 : Lien vers une page interne
  • Étape 3 : Ouvrir un lien dans un nouvel onglet
  • Dépannage

Étape 1 : Lien vers un site externe

Pour créer un lien vers un autre site web, utilisez la balise <a> avec l’attribut href :


<a href="https://www.google.com">Aller sur Google</a>

Ce lien redirigera l'utilisateur vers le site spécifié.

Étape 2 : Lien vers une page interne

Vous pouvez également faire un lien vers une autre page de votre propre site :


<a href="contact.html">Page de contact</a>

Le fichier contact.html doit être dans le même dossier que la page actuelle.

Étape 3 : Ouvrir un lien dans un nouvel onglet

Ajoutez l’attribut target="_blank" pour ouvrir le lien dans un nouvel onglet :


<a href="https://www.wikipedia.org" target="_blank">Wikipedia</a>

Cela permet à l’utilisateur de rester sur votre site tout en visitant l’autre lien.

Dépannage

Vérifiez que l'attribut href contient une URL ou un chemin valide.

Exemples corrects :

  • https://www.example.com
  • page2.html

Assurez-vous qu'aucun style CSS ne supprime les styles par défaut (comme la couleur bleue et le soulignement).

Sinon, ajoutez un style CSS personnalisé :


a {
    color: blue;
    text-decoration: underline;
}
