Étape 1 : Qu’est-ce qu’un attribut ?

1

Un attribut HTML fournit des informations supplémentaires sur un élément.

2

Il est généralement placé dans la balise ouvrante d’un élément HTML.

Étape 2 : Syntaxe des attributs

La syntaxe d’un attribut est toujours :

<balise nom="valeur">Contenu</balise>

Par exemple :

<a href="https://exemple.com">Visitez le site</a>

Ici :

  • href est le nom de l’attribut
  • "https://exemple.com" est sa valeur
  • L’attribut affecte le comportement de l’élément <a>

Étape 3 : Exemples d’attributs HTML courants

Élément Attribut Exemple Description
<a> href <a href="https://site.com">lien</a> Définit la destination du lien
<img> src, alt <img src="logo.png" alt="Logo"> Spécifie l'image et sa description
<input> type, placeholder <input type="text" placeholder="Nom"> Définit le type de champ de formulaire
<button> disabled <button disabled>Envoyer</button> Désactive le bouton

Dépannage

Vérifiez que l’attribut href est bien présent et que l’URL est correcte :

<a href="https://exemple.com">Texte du lien</a>

Assurez-vous que l’attribut src pointe vers le bon fichier et que le chemin est correct.

<img src="chemin/vers/image.jpg" alt="Description">
