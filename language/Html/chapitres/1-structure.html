Étape 1 : Qu’est-ce qu’un document HTML ?

1

HTML (HyperText Markup Language) est le langage utilisé pour structurer les pages web.

2

Un document HTML contient des balises qui organisent le contenu (titres, paragraphes, liens...)

Étape 2 : Structure minimale d’un fichier HTML

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Titre de la page</title>
</head>
<body>
    <h1>Bonjour, monde !</h1>
    <p>Ceci est ma première page HTML.</p>
</body>
</html>

Étape 3 : Détail des balises principales

Balise Description
<!DOCTYPE html> Indique au navigateur qu'il s'agit d’un document HTML5
<html> Balise racine contenant tout le code HTML
<head> Contient les métadonnées (encodage, titre, liens CSS...)
<body> Contient le contenu visible de la page (titres, textes, images...)
<title> Affiche le titre de la page dans l’onglet du navigateur

Dépannage

Vérifiez que votre fichier a bien une structure HTML complète et que la balise <body> contient du contenu.

Assurez-vous que vous avez bien défini l’encodage UTF-8 avec :

<meta charset="UTF-8">
