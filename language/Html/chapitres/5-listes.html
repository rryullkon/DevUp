Listes ordonnées et non ordonnées

Apprenez à structurer des listes en HTML avec les balises <ul>, <ol> et <li>

Sommaire du chapitre

  • Étape 1 : Liste non ordonnée
  • Étape 2 : Liste ordonnée
  • Étape 3 : Listes imbriquées
  • Dépannage

Étape 1 : Liste non ordonnée

Utilisez la balise <ul> pour créer une liste à puces :


<ul>
    <li>Pomme</li>
    <li>Banane</li>
    <li>Orange</li>
</ul>

Chaque élément de liste est défini avec la balise <li>.

Étape 2 : Liste ordonnée

Utilisez <ol> pour créer une liste numérotée :


<ol>
    <li>Étape 1</li>
    <li>Étape 2</li>
    <li>Étape 3</li>
</ol>

La numérotation est générée automatiquement par le navigateur.

Étape 3 : Listes imbriquées

Vous pouvez placer une liste à l'intérieur d'une autre :


<ul>
    <li>Fruits
        <ul>
            <li>Pomme</li>
            <li>Raisin</li>
        </ul>
    </li>
    <li>Légumes</li>
</ul>

Cela permet de créer des sous-catégories visuelles.

Dépannage

Assurez-vous que vos balises <ul> ou <ol> sont bien ouvertes et fermées, et que vos éléments sont dans des <li>.


<ul>
    <li>Élément</li>
</ul>

Vérifiez si un style CSS ne désactive pas l'affichage des puces ou numéros. Par exemple :


ul {
    list-style-type: none;
}

Supprimez cette règle ou remplacez-la par disc pour les puces ou decimal pour les listes numérotées.
