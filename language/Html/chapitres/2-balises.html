Étape 1 : Qu’est-ce qu’une balise ?

1

Une balise est un mot-clé entouré d’angles (< >) utilisé pour marquer une partie du contenu HTML.

2

Il existe des balises ouvrantes <p> et des balises fermantes </p>.

Étape 2 : Différence entre balise et élément

Un élément HTML est constitué d'une balise ouvrante, d'un contenu, puis d'une balise fermante.

<p>Ceci est un paragraphe</p>

Dans cet exemple :

  • <p> est la balise ouvrante
  • Ceci est un paragraphe est le contenu
  • </p> est la balise fermante
  • L’ensemble forme un élément

Étape 3 : Exemples d’éléments HTML

Élément Exemple Description
<h1>...</h1> <h1>Titre principal</h1> Titre de niveau 1
<p>...</p> <p>Un paragraphe.</p> Texte de paragraphe
<a>...</a> <a href="url">lien</a> Lien hypertexte
<img /> <img src="image.png" alt="Description" /> Image (élément autofermant)

Dépannage

Assurez-vous que vos balises sont bien fermées.

<p>Paragraphe</p>

Utilisez les entités HTML pour afficher des balises comme du texte :

&lt;p&gt;Texte&lt;/p&gt;
