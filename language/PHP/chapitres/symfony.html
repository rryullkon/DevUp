Chapitre 60: Symfony

1. Qu'est-ce que Symfony ?

Symfony est un framework PHP robuste et modulaire, utilisé pour développer des applications web professionnelles. Il repose sur une architecture flexible, orientée objet, et s'appuie sur les bonnes pratiques du développement logiciel. De nombreuses grandes entreprises et CMS (comme Drupal) utilisent Symfony.

2. Pourquoi choisir Symfony ?

  • Fiabilité : utilisé dans des projets critiques et à grande échelle.
  • Structure claire : architecture MVC bien définie.
  • Composants réutilisables : chaque fonctionnalité est un composant indépendant.
  • Interopérabilité : compatible avec d'autres bibliothèques PHP via Composer.
  • Documentation et communauté : riches, bien organisées, support professionnel disponible.

3. Prérequis pour Symfony

  • PHP ≥ 8.1
  • Composer installé
  • Serveur local ou Symfony CLI
  • Base de données (MySQL, PostgreSQL, SQLite...)

4. Installation de Symfony

# Installer l'outil CLI Symfony
curl -sS https://get.symfony.com/cli/installer | bash

# Créer un nouveau projet Symfony
symfony new mon-projet --webapp

# Lancer le serveur
cd mon-projet
symfony serve

5. Structure d’un projet Symfony

  • src/ : code source de l’application (contrôleurs, entités, services...)
  • templates/ : vues avec le moteur Twig
  • config/ : configuration de l’application
  • public/ : point d’entrée du site
  • var/ et vendor/ : fichiers générés et dépendances

6. Exemple de route dans Symfony

// src/Controller/AccueilController.php

namespace App\Controller;

use Symfony\Bundle\FrameworkBundle\Controller\AbstractController;
use Symfony\Component\HttpFoundation\Response;
use Symfony\Component\Routing\Annotation\Route;

class AccueilController extends AbstractController
{
    #[Route('/bonjour', name: 'bonjour')]
    public function index(): Response
    {
        return new Response('Bonjour depuis Symfony !');
    }
}

7. Console Symfony

Symfony possède une interface CLI puissante pour générer du code, exécuter des migrations, vider le cache, etc. 

php bin/console list          # Liste des commandes
php bin/console make:controller MonController

8. Conclusion

Symfony est un framework robuste et adapté aux projets complexes. Grâce à ses composants modulaires, son écosystème complet et sa rigueur, il est un excellent choix pour les développeurs souhaitant construire des applications PHP professionnelles.
