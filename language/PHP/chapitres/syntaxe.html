Syntaxe de base

Apprenez la structure fondamentale du code PHP.

Chapitre 4: Syntaxe de base

1. Débuter un script PHP

Un script PHP commence toujours par la balise <?php et se termine par ?>. Tout le code PHP doit se trouver entre ces balises. 

<?php
// Code PHP ici
?>

2. Afficher du texte

Pour afficher du contenu à l’écran, on utilise la fonction echo ou print. 

<?php
echo "Bonjour le monde !";
print "Bienvenue dans le monde PHP";
?>

3. Commentaires

Il existe trois façons d’ajouter des commentaires dans un script PHP : 

// Ceci est un commentaire sur une ligne

# Ceci est aussi un commentaire sur une ligne

/* Ceci est un commentaire
   sur plusieurs lignes */

4. Variables

Les variables en PHP commencent par le signe $ suivi du nom de la variable. Les noms de variables sont sensibles à la casse. 

<?php
$nom = "Alice";
$age = 25;

echo "Nom : $nom";
echo "Âge : $age";
?>

5. Types de données

Voici les principaux types de données en PHP :

  • Chaînes de caractères : "Bonjour"
  • Nombres entiers : 42
  • Nombres à virgule : 3.14
  • Booléens : true ou false
  • Tableaux : array("pomme", "banane")
  • Objets : Instances de classes personnalisées

6. Concaténation de chaînes

Pour coller deux chaînes de caractères, on utilise le point .. 

<?php
$prenom = "Alice";
echo "Bonjour " . $prenom;
?>

7. Fin de ligne et point-virgule

En PHP, chaque instruction se termine par un point-virgule (;). Oublier ce point-virgule peut provoquer des erreurs.

8. Exécution dans un serveur

N'oubliez pas que PHP doit être exécuté via un serveur local (comme XAMPP, WAMP, MAMP, etc.) ou un serveur distant. Ouvrir un fichier PHP directement dans le navigateur ne fonctionnera pas correctement sans serveur.
