Structures de contrôle

Conditions if/else, switch et opérateur ternaire.

Chapitre 8: Structures de contrôle

1. Introduction

Les structures de contrôle permettent à un programme de prendre des décisions. PHP propose plusieurs manières de contrôler le déroulement du code en fonction de conditions.

2. La condition if / else

Elle permet d'exécuter un bloc de code si une condition est vraie. 

<?php
$age = 18;

if ($age >= 18) {
    echo "Vous êtes majeur.";
} else {
    echo "Vous êtes mineur.";
}
?>

Avec elseif

<?php
$note = 15;

if ($note >= 16) {
    echo "Très bien";
} elseif ($note >= 10) {
    echo "Suffisant";
} else {
    echo "Insuffisant";
}
?>

3. La structure switch

Utile quand vous devez comparer une variable à plusieurs valeurs possibles. 

<?php
$jour = "lundi";

switch ($jour) {
    case "lundi":
        echo "Début de semaine";
        break;
    case "vendredi":
        echo "Presque le week-end";
        break;
    case "samedi":
    case "dimanche":
        echo "C'est le week-end !";
        break;
    default:
        echo "Jour ordinaire";
}
?>

4. L'opérateur ternaire

Il permet d’écrire une condition simple en une seule ligne. 

<?php
$connecte = true;
$message = $connecte ? "Bienvenue !" : "Veuillez vous connecter.";
echo $message;
?>

5. Bonnes pratiques

  • Utilisez if/else pour les conditions complexes ou imbriquées.
  • Préférez switch pour les comparaisons multiples sur une même variable.
  • L’opérateur ternaire est pratique pour les cas très simples, mais à éviter si trop complexe.
