Installation de PHP sur macOS

Guide détaillé pour installer PHP sur un Mac avec macOS (Monterey ou version ultérieure)

Sommaire du chapitre

  • Étape 1 : Vérification du PHP intégré
  • Étape 2 : Installation de Homebrew
  • Étape 3 : Installation de PHP avec Homebrew
  • Étape 4 : Configuration et vérification
  • Dépannage

Étape 1 : Vérification du PHP intégré

macOS inclut parfois une version obsolète de PHP. Vérifiez ce que vous avez déjà :

php -v

Si vous voyez une version comme PHP 7.x ou un message d'erreur, passez à l'étape suivante.

Étape 2 : Installation de Homebrew

Homebrew est un gestionnaire de paquets très utile sur macOS. Si vous ne l'avez pas :

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
Note : Après l'installation, ajoutez Homebrew au PATH si nécessaire : 
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"

Étape 3 : Installation de PHP avec Homebrew

Une fois Homebrew installé, installez PHP avec la commande suivante :

brew install php

Par défaut, Homebrew installe la dernière version stable (ex : PHP 8.3).

Étape 4 : Configuration et vérification

Vérifiez que la bonne version de PHP est utilisée :

Terminal
mac@macbook ~ % php -v
PHP 8.3.6 (cli) (built: May 1 2025 10:00:00) (NTS)

Vous pouvez aussi vérifier l'emplacement de PHP avec :

which php

Il doit pointer vers /opt/homebrew/bin/php (ou /usr/local/bin/php selon votre architecture).

Dépannage

Ajoutez manuellement le chemin de Homebrew au début de votre PATH :

export PATH="/opt/homebrew/bin:$PATH"

Ajoutez cette ligne à votre fichier ~/.zshrc ou ~/.bash_profile.

Installez et configurez le module PHP pour Apache :

brew install httpd

Modifiez /opt/homebrew/etc/httpd/httpd.conf pour inclure :

LoadModule php_module /opt/homebrew/opt/php/lib/httpd/modules/libphp.so
AddHandler application/x-httpd-php .php

# Activez le module dans la section DirectoryIndex :
DirectoryIndex index.php index.html

Relancez Apache :

sudo brew services restart httpd
