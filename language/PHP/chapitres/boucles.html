Boucles

for, while, do-while, foreach et contrôle des boucles.

Chapitre 9: Boucles

1. Introduction

Les boucles permettent d’exécuter un bloc de code plusieurs fois tant qu’une condition est vraie ou selon un compteur.

2. La boucle for

Utilisée quand on connaît le nombre d’itérations à effectuer.

<?php
for ($i = 0; $i < 5; $i++) {
    echo "Itération $i
";
}
?>

3. La boucle while

Exécute le bloc tant que la condition est vraie.

<?php
$i = 0;
while ($i < 5) {
    echo "Itération $i
";
    $i++;
}
?>

4. La boucle do-while

Exécute le bloc au moins une fois puis répète tant que la condition est vraie.

<?php
$i = 0;
do {
    echo "Itération $i
";
    $i++;
} while ($i < 5);
?>

5. La boucle foreach

Spécialement conçue pour parcourir les tableaux et objets itérables.

<?php
$fruits = ["pomme", "banane", "cerise"];

foreach ($fruits as $fruit) {
    echo $fruit . "<br>";
}
?>

6. Contrôle des boucles

  • break : sortir immédiatement de la boucle
  • continue : passer directement à l’itération suivante
<?php
for ($i = 0; $i < 10; $i++) {
    if ($i == 5) {
        break; // sort de la boucle dès que $i vaut 5
    }
    if ($i % 2 == 0) {
        continue; // saute les nombres pairs
    }
    echo "Valeur impaire: $i<br>";
}
?>

7. Résumé

Type de boucle Quand l'utiliser ?
for Nombre précis d’itérations
while Condition préalable à chaque itération
do-while Exécution au moins une fois
foreach Parcours d’un tableau ou d’un objet itérable
