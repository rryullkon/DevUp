Extension de classes et polymorphisme.

Chapitre 19: Héritage

1. Qu’est-ce que l’héritage en POO ?

L’héritage permet à une classe (dite classe enfant ou sous-classe) de réutiliser les propriétés et méthodes d’une autre classe (la classe parent ou super-classe). Cela favorise la réutilisation de code et une organisation hiérarchique des objets.

2. Déclaration d’une classe héritée

<?php
class Animal {
    public function crier() {
        echo "L'animal fait un bruit.<br>";
    }
}

class Chien extends Animal {
    public function crier() {
        echo "Le chien aboie : Wouf!<br>";
    }
}

$chien = new Chien();
$chien->crier(); // Affiche "Le chien aboie : Wouf!"
?>

3. Utiliser les propriétés héritées

<?php
class Vehicule {
    public $marque;

    public function afficherMarque() {
        echo "Marque : $this->marque<br>";
    }
}

class Voiture extends Vehicule {
    public $modele;

    public function afficherModele() {
        echo "Modèle : $this->modele<br>";
    }
}

$voiture = new Voiture();
$voiture->marque = "Toyota";
$voiture->modele = "Corolla";
$voiture->afficherMarque();
$voiture->afficherModele();
?>

4. Appel du constructeur parent

<?php
class Personne {
    protected $nom;

    public function __construct($nom) {
        $this->nom = $nom;
    }
}

class Etudiant extends Personne {
    public function afficherNom() {
        echo "Nom : $this->nom<br>";
    }
}

$etudiant = new Etudiant("Alice");
$etudiant->afficherNom();
?>

5. Le polymorphisme

Le polymorphisme permet d’utiliser une même méthode avec un comportement différent selon la classe. Cela permet de généraliser les traitements tout en ayant des comportements spécialisés. 

<?php
class Forme {
    public function dessiner() {
        echo "Dessiner une forme générique.<br>";
    }
}

class Cercle extends Forme {
    public function dessiner() {
        echo "Dessiner un cercle.<br>";
    }
}

class Carre extends Forme {
    public function dessiner() {
        echo "Dessiner un carré.<br>";
    }
}

$formes = [new Cercle(), new Carre(), new Forme()];

foreach ($formes as $forme) {
    $forme->dessiner();
}
?>

6. Résumé

  • Une classe peut hériter d’une autre via le mot-clé extends
  • Le polymorphisme permet d’appeler des méthodes avec un comportement propre à chaque classe
  • On peut réécrire une méthode héritée (surcharge)
  • Le constructeur du parent peut être appelé depuis l’enfant
