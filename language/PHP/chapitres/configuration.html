Configuration de PHP

Comprendre et modifier le fichier php.ini pour optimiser PHP.

Chapitre 3: Configuration de PHP

1. Qu'est-ce que le fichier php.ini ?

Le fichier php.ini est le fichier principal de configuration de PHP. Il permet de définir le comportement de PHP sur votre serveur. Lors de l’exécution d’un script PHP, PHP lit ce fichier pour appliquer les réglages définis.

2. Où se trouve php.ini ?

L’emplacement du fichier php.ini peut varier selon le système d’exploitation et la méthode d’installation :

  • Windows (XAMPP/WAMP) : Généralement dans le dossier php/ ou apache/bin/.
  • macOS (Homebrew) : Utilisez php --ini pour connaître le chemin exact.
  • Linux (Ubuntu) : Le fichier se trouve souvent dans /etc/php/<version>/apache2/php.ini.

3. Modifier php.ini

Pour modifier php.ini, ouvrez-le avec un éditeur de texte avec les droits administrateur. Voici quelques directives utiles :

  • display_errors = On – Affiche les erreurs à l’écran (utile en développement).
  • memory_limit = 256M – Définit la limite de mémoire utilisée par un script PHP.
  • upload_max_filesize = 20M – Taille maximale pour un fichier téléchargé.
  • post_max_size = 25M – Taille maximale des données envoyées via POST.
  • max_execution_time = 30 – Temps maximum d’exécution d’un script (en secondes).

4. Appliquer les modifications

Après avoir modifié le fichier php.ini, il est nécessaire de redémarrer le serveur web (Apache, Nginx, etc.) pour que les changements soient pris en compte.

5. Vérifier la configuration

Utilisez le script suivant pour afficher la configuration PHP actuelle : 

<?php
phpinfo();
?>

Ce script affiche tous les paramètres actuels, y compris ceux définis dans php.ini.
