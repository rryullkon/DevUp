Fonctions récursives

Principes et exemples de récursivité en PHP.

Chapitre 15: Fonctions récursives

1. Qu'est-ce qu'une fonction récursive ?

Une fonction récursive est une fonction qui s'appelle elle-même, directement ou indirectement. Elle permet de résoudre certains problèmes en divisant la tâche en sous-tâches similaires.

2. Conditions importantes

  • Une condition d'arrêt pour éviter une récursion infinie.
  • Un appel récursif vers la même fonction avec un paramètre modifié.

3. Exemple simple : factorielle

<?php
function factorielle($n) {
    if ($n <= 1) {
        return 1;  // condition d'arrêt
    } else {
        return $n * factorielle($n - 1);  // appel récursif
    }
}

echo factorielle(5); // Affiche 120
?>

4. Exemple avancé : calcul de la suite de Fibonacci

<?php
function fibonacci($n) {
    if ($n === 0) {
        return 0;
    } elseif ($n === 1) {
        return 1;
    } else {
        return fibonacci($n - 1) + fibonacci($n - 2);
    }
}

echo fibonacci(7); // Affiche 13
?>

5. Utilité et limites

La récursivité est idéale pour traiter des structures arborescentes ou fractales (ex. arbres, dossiers, etc.). Cependant, elle peut entraîner une consommation mémoire importante et un risque de dépassement de pile (stack overflow) si la condition d'arrêt n'est pas correctement définie.

6. Résumé

  • Une fonction récursive s’appelle elle-même
  • Elle doit toujours avoir une condition d’arrêt
  • Permet de simplifier certains algorithmes complexes
  • Attention aux appels excessifs pour éviter les erreurs
