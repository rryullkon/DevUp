PHP et HTML

Intégration de PHP dans les pages HTML.

Chapitre 31: PHP et HTML

1. Introduction

PHP est souvent utilisé pour générer du contenu HTML dynamiquement. Il s'intègre directement dans les fichiers HTML via des balises spéciales.

2. Syntaxe d’intégration

Le code PHP est inclus dans une page HTML entre les balises <?php ?> : 

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Page PHP</title>
</head>
<body>

<h1>Bienvenue</h1>

<?php
    echo "<p>Bonjour depuis PHP !</p>";
?>

</body>
</html>

3. Affichage dynamique

On peut insérer des variables, conditions, boucles, etc., pour créer du contenu adapté à chaque visite : 

<?php
$heure = date("H");
if ($heure < 12) {
    echo "<p>Bonjour !</p>";
} else {
    echo "<p>Bonne après-midi !</p>";
}
?>

4. Courte syntaxe

Pour afficher rapidement une variable, on peut utiliser la syntaxe abrégée <?= $variable ?> : 

<?php $nom = "Alice"; ?>
<p>Bonjour  !</p>

5. Mélanger PHP et HTML

Le PHP peut être inséré n’importe où dans la page HTML, permettant un mélange souple :

<ul>
    <?php
    $fruits = ["Pomme", "Banane", "Cerise"];
    foreach ($fruits as $fruit) {
        echo "<li>$fruit</li>";
    }
    ?>
</ul>

6. Résumé

  • PHP s’intègre directement dans le HTML via <?php ?>
  • Syntaxe abrégée <?= ?> pour afficher rapidement
  • Possibilité de générer du contenu dynamique avec variables, conditions et boucles
  • Idéal pour créer des pages web dynamiques et interactives
