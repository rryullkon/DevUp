Installation de PHP sur Windows

Guide complet pour installer PHP correctement sur votre système Windows

Sommaire du chapitre

  • Étape 1 : Téléchargement
  • Étape 2 : Extraction et configuration
  • Étape 3 : Ajout au PATH
  • Étape 4 : Vérification
  • Dépannage

Étape 1 : Téléchargement

Commencez par télécharger PHP depuis le site officiel :

Note importante : Téléchargez PHP uniquement depuis le site officiel pour éviter toute version corrompue.
1

Rendez-vous sur php.net/downloads.php

2

Téléchargez la version "Windows x64 Thread Safe" en format zip

3

Extrayez l'archive dans un dossier, par exemple C:\php

Étape 2 : Extraction et configuration

Après avoir extrait les fichiers :

1

Renommez le fichier php.ini-development en php.ini

2

Ouvrez php.ini avec un éditeur de texte (ex: Notepad++)

3

Activez les extensions nécessaires en décommentant les lignes (par ex. : extension=mysqli)

; Exemple : Activer MySQLi
extension=mysqli
extension=pdo_mysql

Étape 3 : Ajout au PATH

Pour pouvoir utiliser PHP depuis n’importe quel dossier :

Important : Il est essentiel d’ajouter le dossier PHP au PATH système.
1

Ouvrez les Paramètres système avancés > Variables d’environnement

2

Dans la section Path, ajoutez le chemin vers le dossier PHP (ex: C:\php)

3

Redémarrez le terminal pour appliquer les changements

Étape 4 : Vérification

Testez si PHP est bien installé :

Command Prompt
C:\> php -v
PHP 8.3.0 (cli) (built: Nov 21 2024 12:00:00) ( ZTS Visual C++ 2019 x64 )

Si une version s'affiche, l'installation a réussi !

Dépannage

Vérifiez que PHP est bien ajouté au PATH :

  1. Allez dans les Variables d’environnement
  2. Vérifiez que le chemin vers PHP est présent (ex: C:\php)
  3. Fermez et rouvrez votre terminal

En cas de problème avec des extensions :

  1. Vérifiez que l’extension est bien décommentée (pas de ; en début de ligne)
  2. Vérifiez que le fichier php.ini est dans le bon dossier
  3. Redémarrez le terminal après modification
