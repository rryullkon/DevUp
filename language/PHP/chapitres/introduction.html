Introduction à PHP

Découvrez ce qu'est PHP, son histoire et ses cas d'utilisation.

Chapitre 1: Introduction à PHP

1. Qu'est-ce que PHP ?

PHP (Hypertext Preprocessor) est un langage de programmation côté serveur principalement utilisé pour créer des sites web dynamiques.
Il a été créé en 1994 par Rasmus Lerdorf et s'est largement répandu grâce à sa facilité d'intégration avec le HTML.

2. À quoi sert PHP ?

PHP est utilisé dans de nombreux contextes liés au web :

  • Développement de sites web dynamiques
  • Création de systèmes de gestion de contenu (comme WordPress, Joomla)
  • Connexion à des bases de données (MySQL, PostgreSQL)
  • Création d'APIs REST
  • Gestion de formulaires et sessions

3. Pourquoi choisir PHP ?

Les avantages de PHP :

  1. Facile à apprendre et à intégrer avec HTML
  2. Compatible avec la majorité des serveurs web
  3. Large communauté et documentation abondante
  4. Nombreux frameworks disponibles (Laravel, Symfony)

4. Exemple de code PHP

Voici un exemple simple de script PHP qui affiche un message et effectue une addition : 

<?php
// Ceci est un commentaire
echo "Bonjour le monde !";

$a = 5;
$b = 3;
$somme = $a + $b;
echo "La somme de $a et $b vaut $somme";
?>

5. Premiers pas

Pour commencer avec PHP :

  1. Installez un serveur local comme XAMPP ou WAMP
  2. Créez des fichiers .php dans le dossier de votre serveur
  3. Utilisez un éditeur comme VS Code ou Sublime Text
  4. Apprenez les bases à travers des tutoriels en ligne
