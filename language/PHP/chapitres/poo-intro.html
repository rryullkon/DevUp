Introduction à la POO

Concepts de base : classes, objets, propriétés, méthodes.

Chapitre 16: Introduction à la POO

1. Qu'est-ce que la POO ?

La Programmation Orientée Objet (POO) est un paradigme de programmation qui organise le code autour d’objets. Ces objets regroupent des données (propriétés) et des comportements (méthodes).

2. Classe

Une classe est un modèle ou un plan qui définit les propriétés et méthodes que ses objets auront. 

<?php
class Voiture {
    public $marque;
    public $couleur;

    public function demarrer() {
        echo "La voiture démarre.";
    }
}
?>

3. Objet

Un objet est une instance d’une classe. Chaque objet a ses propres valeurs pour les propriétés. 

<?php
$maVoiture = new Voiture();
$maVoiture->marque = "Peugeot";
$maVoiture->couleur = "Rouge";
$maVoiture->demarrer(); // Affiche "La voiture démarre."
?>

4. Propriétés

Les propriétés sont des variables définies dans une classe, décrivant l’état d’un objet.

5. Méthodes

Les méthodes sont des fonctions définies dans une classe, représentant les actions qu’un objet peut effectuer.

6. Visibilité

Les propriétés et méthodes peuvent avoir des niveaux de visibilité :

  • public : accessibles partout
  • protected : accessibles dans la classe et ses héritiers
  • private : accessibles uniquement dans la classe

7. Exemple complet

<?php
class Personne {
    public $nom;
    public $age;

    public function sePresenter() {
        echo "Je m'appelle $this->nom et j'ai $this->age ans.<br>";
    }
}

$personne1 = new Personne();
$personne1->nom = "Alice";
$personne1->age = 30;
$personne1->sePresenter();

$personne2 = new Personne();
$personne2->nom = "Bob";
$personne2->age = 25;
$personne2->sePresenter();
?>

8. Résumé

  • La classe est un modèle pour créer des objets
  • Les objets ont des propriétés (données) et des méthodes (actions)
  • La POO facilite la structuration, la réutilisation et la maintenance du code
