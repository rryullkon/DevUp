Types de données

Les différents types scalaires, composés et spéciaux en PHP.

Chapitre 6: Types de données

1. Introduction

PHP gère plusieurs types de données que l’on peut classer en trois grandes catégories :

  • Types scalaires
  • Types composés
  • Types spéciaux

2. Les types scalaires

Ce sont les types de base les plus simples :

  • string (chaîne de caractères)
  • int (entier)
  • float (nombre à virgule flottante)
  • bool (booléen : true ou false)
<?php
$nom = "Alice";     // string
$age = 25;          // int
$taille = 1.68;     // float
$inscrit = true;    // bool
?>

3. Les types composés

Ces types permettent de regrouper plusieurs valeurs :

a) Tableaux (array)

Un tableau peut contenir plusieurs valeurs, accessibles via des clés (indices).

<?php
$fruits = array("pomme", "banane", "cerise");
echo $fruits[0]; // Affiche "pomme"
?>

b) Objets (object)

Un objet est une instance d'une classe, utilisé en programmation orientée objet.

<?php
class Personne {
    public $nom = "Jean";
}

$p = new Personne();
echo $p->nom; // Affiche "Jean"
?>

4. Les types spéciaux

PHP propose également des types pour des cas particuliers :

  • null : une variable sans valeur
  • resource : fait référence à des ressources externes (ex : connexion à une base de données, fichier ouvert, etc.)
<?php
$variable = null;

$fichier = fopen("monfichier.txt", "r"); // type resource
?>

5. Vérifier ou forcer un type

PHP propose des fonctions natives pour tester ou convertir les types :

  • is_string($var), is_int($var), is_array($var), etc.
  • Forcer un type : (int), (float), (string)
<?php
$nombre = "42";
$nombre = (int) $nombre; // conversion en entier
echo is_int($nombre);    // true (1)
?>

6. Fonction gettype()

Pour afficher dynamiquement le type d’une variable : 

<?php
$valeur = 3.14;
echo gettype($valeur); // Affiche "double"
?>
