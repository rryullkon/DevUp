Opérateurs

Opérateurs arithmétiques, de comparaison, logiques et autres.

Chapitre 7: Opérateurs

1. Qu'est-ce qu'un opérateur ?

Un opérateur permet d’effectuer des opérations sur des variables ou des valeurs. PHP propose plusieurs catégories d’opérateurs selon le type d’opération souhaitée.

2. Opérateurs arithmétiques

Utilisés pour les calculs mathématiques :

  • + : addition
  • - : soustraction
  • * : multiplication
  • / : division
  • % : modulo (reste de la division entière)
<?php
$a = 10;
$b = 3;

echo $a + $b; // 13
echo $a % $b; // 1
?>

3. Opérateurs d'affectation

Permettent d’assigner une valeur à une variable :

  • = : affectation simple
  • += : addition puis affectation
  • -=, *=, /=, %= : même logique
<?php
$x = 5;
$x += 3; // équivaut à $x = $x + 3;
echo $x; // 8
?>

4. Opérateurs de comparaison

Utilisés dans les conditions (if, while, etc.) :

  • == : égalité (valeur)
  • === : égalité stricte (valeur et type)
  • != ou <> : différence
  • !== : différence stricte
  • >, < : supérieur, inférieur
  • >=, <= : supérieur ou égal, inférieur ou égal
<?php
$a = 5;
$b = "5";

var_dump($a == $b);  // true (valeurs égales)
var_dump($a === $b); // false (types différents)
?>

5. Opérateurs logiques

Permettent de combiner plusieurs conditions :

  • && ou and : ET logique
  • || ou or : OU logique
  • ! : NON logique (négation)
<?php
$age = 20;
$inscrit = true;

if ($age >= 18 && $inscrit) {
    echo "Accès autorisé";
}
?>

6. Opérateur de concaténation

Permet de coller plusieurs chaînes de caractères :

<?php
$prenom = "Alice";
$nom = "Dupont";

echo $prenom . " " . $nom; // Affiche : Alice Dupont
?>

7. Opérateur ternaire

Opérateur conditionnel compact :

<?php
$age = 17;
$message = ($age >= 18) ? "Majeur" : "Mineur";
echo $message; // Affiche "Mineur"
?>

8. Opérateur null coalescent (PHP 7+)

Permet de fournir une valeur par défaut si une variable est null ou non définie :

<?php
$nom = $_GET['nom'] ?? "Invité";
echo "Bonjour $nom";
?>
