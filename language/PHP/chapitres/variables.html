Variables et constantes

Déclaration, typage et portée des variables en PHP.

Chapitre 5: Variables et constantes

1. Qu'est-ce qu'une variable ?

Une variable est un espace en mémoire utilisé pour stocker une information temporaire (nombre, texte, tableau, etc.). En PHP, toute variable commence par le symbole $, suivi d'un nom. 

<?php
$nom = "Alice";
$age = 30;
$estMembre = true;
?>

2. Règles de nommage

  • Le nom d'une variable commence toujours par $
  • Il doit commencer par une lettre ou un underscore (_)
  • Il ne doit pas contenir d'espaces ni de caractères spéciaux
  • Les noms sont sensible à la casse ($Nom ≠ $nom)

3. Typage des variables

PHP est un langage à typage dynamique, ce qui signifie que le type d'une variable est détecté automatiquement selon la valeur que vous lui attribuez. 

<?php
$nombre = 10;          // entier
$prix = 19.99;         // flottant
$nom = "Jean";         // chaîne de caractères
$estValide = false;    // booléen
?>

Forcer un type (conversion explicite)

Il est possible de forcer une variable à changer de type : 

<?php
$valeur = "100";
$valeur = (int) $valeur; // convertie en entier
?>

4. Tester le type d'une variable

PHP propose des fonctions pour vérifier le type d’une variable :

  • is_int($var) — est-ce un entier ?
  • is_string($var) — est-ce une chaîne ?
  • is_bool($var) — est-ce un booléen ?
  • is_array($var) — est-ce un tableau ?

5. Portée des variables

La portée d’une variable détermine où elle est accessible dans votre script.

  • Locale : définie à l’intérieur d’une fonction
  • Globale : accessible dans tout le script sauf à l’intérieur des fonctions
  • Super globale : variables accessibles partout (ex : $_GET, $_POST)
<?php
$nom = "Alice"; // variable globale

function direBonjour() {
    global $nom; // accès à la variable globale
    echo "Bonjour $nom";
}

direBonjour();
?>

6. Qu'est-ce qu'une constante ?

Une constante est une valeur qui ne peut pas être modifiée une fois définie. Elle n’utilise pas le symbole $. 

<?php
define("PI", 3.14);
echo PI; // Affiche 3.14
?>

Depuis PHP 7, on peut aussi utiliser const pour définir des constantes (principalement à l’intérieur des classes). 

<?php
const SITE_NAME = "MonSite";
echo SITE_NAME;
?>

7. Bonnes pratiques

  • Utilisez des noms clairs et significatifs : $utilisateur plutôt que $u
  • Évitez les majuscules dans les noms de variables (réservées souvent aux constantes)
  • Respectez la portée des variables pour éviter les conflits
  • Utilisez const ou define() pour les valeurs fixes
