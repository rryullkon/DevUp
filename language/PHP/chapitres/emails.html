Envoi d'emails avec PHP

PHPMailer et envoi d'emails en PHP.

Chapitre 77: Envoi d'emails

1. Introduction

Envoyer des emails depuis une application PHP est une fonctionnalité essentielle pour les notifications, confirmations, ou newsletters. La bibliothèque PHPMailer simplifie grandement cette tâche en gérant SMTP, pièces jointes, HTML, et bien plus.

2. Installation

La méthode recommandée est d’utiliser Composer : 

composer require phpmailer/phpmailer

3. Exemple simple d’envoi d’email

<?php
use PHPMailer\PHPMailer\PHPMailer;
use PHPMailer\PHPMailer\Exception;

require 'vendor/autoload.php';

$mail = new PHPMailer(true);

try {
    // Configuration du serveur SMTP
    $mail->isSMTP();
    $mail->Host = 'smtp.example.com';  // Remplacez par votre serveur SMTP
    $mail->SMTPAuth = true;
    $mail->Username = 'votre_email@example.com'; // SMTP username
    $mail->Password = 'votre_mot_de_passe';      // SMTP password
    $mail->SMTPSecure = 'tls';                    // TLS ou SSL
    $mail->Port = 587;                            // Port SMTP

    // Destinataires
    $mail->setFrom('from@example.com', 'Votre Nom');
    $mail->addAddress('destinataire@example.com', 'Nom Destinataire');

    // Contenu de l’email
    $mail->isHTML(true);
    $mail->Subject = 'Test d’envoi d’email avec PHPMailer';
    $mail->Body    = '<p>Ceci est un email >HTML envoyé avec PHPMailer.</p>';
    $mail->AltBody = 'Ceci est un email texte.';

    $mail->send();
    echo 'Email envoyé avec succès';
} catch (Exception $e) {
    echo "Erreur lors de l'envoi de l'email: {$mail->ErrorInfo}";
}
?>

4. Conseils

  • Configurez correctement votre serveur SMTP et assurez-vous que les ports sont ouverts.
  • Utilisez des emails d’expéditeur valides pour éviter le spam.
  • PHPMailer supporte aussi l’envoi via mail(), SMTP avec authentification, et OAuth.
  • Consultez la documentation officielle pour des fonctionnalités avancées.
