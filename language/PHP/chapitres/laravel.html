Chapitre 59: Laravel

1. Qu'est-ce que Laravel ?

Laravel est un framework PHP moderne, puissant et élégant qui facilite le développement d’applications web robustes. Il repose sur une architecture MVC (Modèle-Vue-Contrôleur) et propose de nombreux outils intégrés comme le routage, l’ORM Eloquent, la validation, l’authentification et bien plus.

2. Pourquoi choisir Laravel ?

  • Simplicité : syntaxe claire et intuitive.
  • Fonctionnalités intégrées : système de migration, tâches planifiées, gestion des mails, etc.
  • Communauté active : grande base d’utilisateurs et de ressources.
  • Performance : optimisé pour le développement rapide d’applications modernes.
  • Écosystème riche : outils comme Laravel Forge, Vapor, Nova, Breeze, Jetstream, etc.

3. Prérequis pour utiliser Laravel

  • PHP ≥ 8.1
  • Composer (gestionnaire de dépendances PHP)
  • Serveur web local (Apache, Nginx ou Laravel Sail)
  • Base de données (MySQL, SQLite, PostgreSQL...)

4. Installation de Laravel

# Installation via Composer
composer create-project laravel/laravel mon-projet

# Ou via l’installeur Laravel
composer global require laravel/installer
laravel new mon-projet

# Lancer le serveur
php artisan serve

5. Structure de base d’un projet Laravel

  • routes/web.php : définition des routes web.
  • app/Http/Controllers : logique métier.
  • resources/views : vues Blade (HTML).
  • app/Models : modèles Eloquent.
  • database/migrations : structure de la base de données.

6. Exemple de route simple

Route::get('/bonjour', function () {
    return 'Bonjour depuis Laravel !';
});

7. Artisan : l’outil en ligne de commande

Artisan est l’interface CLI de Laravel. Il permet de créer des contrôleurs, exécuter les migrations, lancer des tests, etc. 

php artisan list      # Affiche les commandes disponibles
php artisan make:controller MonController

8. Conclusion

Laravel est un framework complet et puissant qui permet de développer des applications PHP rapidement et efficacement. Son écosystème, sa documentation et sa communauté en font un choix de référence pour les développeurs PHP modernes.
