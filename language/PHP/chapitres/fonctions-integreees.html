Fonctions intégrées

Les fonctions les plus utiles de la bibliothèque standard PHP.

Chapitre 11: Fonctions intégrées

1. Introduction

PHP propose des centaines de fonctions intégrées pour manipuler les chaînes, les tableaux, les fichiers, etc. Voici une sélection des plus courantes et utiles.

2. Fonctions sur les chaînes de caractères

  • strlen($str) : longueur d’une chaîne
  • strtoupper($str) : convertir en majuscules
  • strtolower($str) : convertir en minuscules
  • substr($str, $start, $length) : extraire une sous-chaîne
  • strpos($str, $search) : position de la première occurrence
  • str_replace($search, $replace, $str) : remplacer dans une chaîne

3. Fonctions sur les tableaux

  • count($array) : nombre d’éléments
  • array_merge($arr1, $arr2) : fusionner des tableaux
  • array_keys($array) : obtenir les clés
  • array_values($array) : obtenir les valeurs
  • in_array($value, $array) : vérifier la présence d’une valeur

4. Fonctions de manipulation de fichiers

  • file_get_contents($filename) : lire un fichier en entier
  • file_put_contents($filename, $data) : écrire dans un fichier
  • fopen(), fread(), fwrite(), fclose() : accès fichier plus précis
  • unlink($filename) : supprimer un fichier

5. Fonctions utiles diverses

  • date($format) : afficher la date/heure formatée
  • empty($var) : teste si une variable est vide
  • isset($var) : teste si une variable est définie
  • var_dump($var) : affiche le contenu détaillé d’une variable

6. Exemple d’utilisation

<?php
$texte = "Bonjour le monde";
echo strtoupper($texte); // BONJOUR LE MONDE

$fruits = ["pomme", "banane", "cerise"];
if (in_array("banane", $fruits)) {
    echo "Banane est dans le tableau.";
}

$fichier = "exemple.txt";
file_put_contents($fichier, "Contenu du fichier");
echo file_get_contents($fichier);
?>
