Installation de PHP sur Linux

Guide complet pour installer PHP correctement sur votre système Linux (Ubuntu/Debian)

Sommaire du chapitre

  • Étape 1 : Mise à jour du système

Étape 1 : Mise à jour du système

Avant toute installation, commencez par mettre à jour les paquets existants :

sudo apt update
sudo apt upgrade
Note : Ces commandes assurent que votre gestionnaire de paquets est à jour.

Étape 2 : Installation de PHP

Installez PHP avec la commande suivante :

sudo apt install php

Cela installera la version stable de PHP disponible dans vos dépôts.

Besoin d’une version spécifique ? Utilisez un PPA comme ondrej/php pour accéder à d'autres versions. 
sudo add-apt-repository ppa:ondrej/php
sudo apt update
sudo apt install php8.3

Étape 3 : Installation des extensions

Ajoutez les extensions dont vous avez besoin :

sudo apt install php-mysql php-cli php-curl php-xml php-mbstring

Ces extensions sont souvent requises pour les frameworks comme Laravel ou Symfony.

Étape 4 : Vérification

Vérifiez si PHP est bien installé :

Terminal
user@ubuntu:~$ php -v
PHP 8.3.0 (cli) (built: Nov 23 2024 12:00:00) ( NTS )

Un message comme ci-dessus confirme le bon fonctionnement de PHP.

Dépannage

Essayez les étapes suivantes :

  1. Vérifiez que PHP est bien installé : which php
  2. Ajoutez manuellement le chemin de PHP à votre PATH
  3. Relancez votre terminal ou session

Assurez-vous que le module PHP pour Apache est installé :

sudo apt install libapache2-mod-php
sudo systemctl restart apache2
