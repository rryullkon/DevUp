Chapitre 2: Installation de PHP

1. Introduction

Avant de pouvoir développer des applications en PHP, il est nécessaire d’installer l’interpréteur PHP sur votre système. PHP est compatible avec les trois principaux systèmes d'exploitation : Windows, Linux et macOS.

2. Installation sur Windows

  1. Téléchargez XAMPP ou WampServer.
  2. Installez l’un de ces serveurs qui intègrent PHP, Apache et MySQL.
  3. Lancez le panneau de contrôle et démarrez le serveur Apache.
  4. Placez vos fichiers PHP dans le dossier htdocs de XAMPP ou www de WAMP.

3. Installation sur macOS

  1. Installez Homebrew si ce n’est pas déjà fait.
  2. Ouvrez le Terminal et exécutez la commande : brew install php
  3. Vérifiez l’installation avec : php -v

4. Installation sur Linux (Ubuntu)

  1. Ouvrez un terminal.
  2. Mettez à jour les paquets : sudo apt update
  3. Installez PHP avec la commande : sudo apt install php
  4. Vérifiez l’installation avec : php -v

5. Tester l'installation

Créez un fichier test.php contenant le code suivant : 

<?php
phpinfo();
?>

Placez ce fichier dans le dossier approprié de votre serveur (par exemple, htdocs avec XAMPP), puis accédez-y via votre navigateur : http://localhost/test.php.
