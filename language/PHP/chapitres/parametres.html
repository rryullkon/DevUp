Paramètres et valeurs de retour

Passage par valeur/référence, paramètres optionnels.

Chapitre 13: Paramètres et valeurs de retour

1. Passage par valeur

Par défaut, les paramètres d’une fonction sont passés par valeur : une copie de la variable est utilisée, donc la variable d’origine ne change pas. 

<?php
function ajouterCinq($nombre) {
    $nombre += 5;
}

$x = 10;
ajouterCinq($x);
echo $x; // Affiche 10, car $x n'a pas été modifié
?>

2. Passage par référence

Pour modifier la variable d’origine, il faut passer le paramètre par référence en ajoutant un & devant le nom du paramètre. 

<?php
function ajouterCinqParRef(&$nombre) {
    $nombre += 5;
}

$x = 10;
ajouterCinqParRef($x);
echo $x; // Affiche 15, car $x a été modifié
?>

3. Valeur de retour

Une fonction peut renvoyer une valeur avec return, qui peut être stockée dans une variable. 

<?php
function addition($a, $b) {
    return $a + $b;
}

$resultat = addition(3, 7);
echo $resultat; // Affiche 10
?>

4. Paramètres optionnels

On peut définir des valeurs par défaut pour les paramètres, ce qui les rend optionnels lors de l’appel. 

<?php
function saluer($nom = "Invité") {
    echo "Bonjour, $nom !";
}

saluer();        // Affiche "Bonjour, Invité !"
saluer("Alice"); // Affiche "Bonjour, Alice !"
?>

5. Paramètres obligatoires et optionnels

Les paramètres obligatoires doivent être placés avant les paramètres optionnels. 

<?php
function presentation($prenom, $age = null) {
    echo "Je m'appelle $prenom";
    if ($age !== null) {
        echo " et j'ai $age ans.";
    }
}
presentation("Paul");        // OK
presentation("Paul", 25);    // OK
?>

6. Résumé

  • Passage par valeur : copie, variable originale inchangée
  • Passage par référence : modification directe de la variable
  • Paramètres optionnels avec valeurs par défaut
  • La fonction peut retourner une valeur avec return
