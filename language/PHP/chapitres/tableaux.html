Tableaux

Tableaux indexés, associatifs et multidimensionnels.

Chapitre 10: Tableaux

1. Introduction

Les tableaux en PHP permettent de stocker plusieurs valeurs dans une seule variable. Ils peuvent être indexés numériquement, associatifs ou multidimensionnels.

2. Tableaux indexés

Les clés sont des nombres entiers automatiques commençant à 0.

<?php
$fruits = ["pomme", "banane", "cerise"];
echo $fruits[1]; // affiche "banane"
?>

3. Tableaux associatifs

Les clés sont des chaînes de caractères personnalisées.

<?php
$personne = [
    "nom" => "Dupont",
    "prenom" => "Alice",
    "age" => 30
];
echo $personne["prenom"]; // affiche "Alice"
?>

4. Tableaux multidimensionnels

Tableaux contenant d'autres tableaux en valeurs.

<?php
$contacts = [
    [
        "nom" => "Dupont",
        "prenom" => "Alice"
    ],
    [
        "nom" => "Martin",
        "prenom" => "Paul"
    ]
];
echo $contacts[1]["nom"]; // affiche "Martin"
?>

5. Parcourir un tableau

On utilise souvent la boucle foreach pour parcourir les tableaux.

<?php
foreach ($fruits as $fruit) {
    echo $fruit . "<br>";
}

foreach ($personne as $cle => $valeur) {
    echo "$cle : $valeur<br>";
}
?>

6. Fonctions utiles sur les tableaux

  • count($tableau) : nombre d’éléments
  • array_keys($tableau) : récupère les clés
  • array_values($tableau) : récupère les valeurs
  • in_array($valeur, $tableau) : teste la présence d’une valeur
