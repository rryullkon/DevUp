Fonctions utilisateur

Création et utilisation de vos propres fonctions.

Chapitre 12: Fonctions utilisateur

1. Introduction

Les fonctions utilisateur permettent de regrouper un bloc de code réutilisable, ce qui facilite la maintenance et la lisibilité du code.

2. Déclaration d'une fonction

La syntaxe de base :

<?php
function nomDeLaFonction() {
    // instructions
}
?>

3. Exemple simple

<?php
function direBonjour() {
    echo "Bonjour !<br>";
}

direBonjour(); // Appel de la fonction
?>

4. Fonctions avec paramètres

Les fonctions peuvent accepter des arguments pour personnaliser leur comportement.

<?php
function saluer($nom) {
    echo "Bonjour, $nom !<br>";
}

saluer("Alice");
saluer("Bob");
?>

5. Valeur de retour

Une fonction peut retourner une valeur grâce à return.

<?php
function addition($a, $b) {
    return $a + $b;
}

$somme = addition(5, 3);
echo "La somme est $somme.";
?>

6. Paramètres avec valeur par défaut

<?php
function saluerAvecDefaut($nom = "Invité") {
    echo "Bonjour, $nom !<br>";
}

saluerAvecDefaut(); // Bonjour, Invité !
saluerAvecDefaut("Alice"); // Bonjour, Alice !
?>

7. Portée des variables

Les variables déclarées à l'intérieur d'une fonction ne sont pas accessibles à l'extérieur (portée locale).

<?php
function testPortee() {
    $message = "Variable locale";
    echo $message;
}

testPortee();
// echo $message; // Erreur : $message n'existe pas ici
?>

8. Variables globales

Pour accéder à une variable globale dans une fonction, utilisez global ou le tableau $GLOBALS.

<?php
$message = "Variable globale";

function afficherMessage() {
    global $message;
    echo $message;
}

afficherMessage();
?>
